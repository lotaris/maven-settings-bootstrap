Constructor

At the moment, only these options are supported:

  • configFile: A path to the msb configuration file for the generation process
    default: ~/.m2/msb.yml
  • bootstrapFile: A path to the bootstrap setting.xml file
    default: msb-settings.xml
  • outputFile: A path to the place where the settings.xml will be generated. The name of the file itself must be part of the path.
    default: settings.xml
